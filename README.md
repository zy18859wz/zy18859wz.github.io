# Weizhe Niu personal webpage

Static personal webpage for GitHub Pages.

## Files

- `index.html` — main webpage content.
- `styles.css` — page styling.
- `master-thesis.pdf` — MASt project PDF linked from the Education section.

## Deploy

Create or use the GitHub repository:

```text
zy18859wz.github.io
```

Upload these files directly into the repository root:

```text
index.html
styles.css
master-thesis.pdf
README.md
```

The webpage should then be available at:

```text
https://zy18859wz.github.io/
```

If needed, check `Settings -> Pages` and set:

```text
Source: Deploy from a branch
Branch: main
Folder: / root
```

## Updating later

Edit `index.html` to update employment, education, papers, thesis links, contact information, or research interests.

Edit `styles.css` to change font size, colours, spacing, or layout.
