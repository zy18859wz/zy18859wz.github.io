# Weizhe Niu personal webpage

This is a small static academic webpage suitable for GitHub Pages.

## Files

- `index.html` — main webpage
- `styles.css` — visual styling

## Deploy on GitHub Pages

1. Create a GitHub repository named `YOUR-GITHUB-USERNAME.github.io`.
2. Upload `index.html` and `styles.css` to the repository root.
3. In `index.html`, replace `https://YOUR-GITHUB-USERNAME.github.io/` in the JSON-LD block with your actual GitHub Pages URL.
4. Commit and push.
5. Visit `https://YOUR-GITHUB-USERNAME.github.io/`.

## SEO notes

- Keep the page title beginning with `Weizhe Niu`.
- Keep your name, affiliation, research area, and arXiv links in plain text.
- Add the URL to your arXiv author profile, Google Scholar profile, ORCID, university profile, and email signature when available.
- Google indexing usually improves once reputable pages link to this site.
